# Sprzęt

Większość sprzedawców jest skłonna do targowania się o ceny, a w niektórych krajach nawet się tego spodziewają to ogólnie rzecz biorąc, można zakupić popularne przedmioty po atrakcyjnych cenach, takich jak te podane w poniższej tabeli(jeśli jakiegoś przedmiotu nie ma w tabeli lub gracz chce coś na zamówienie wtedy MG musi ustalić odpowiednią cenę do danego przedmiotu):

### Ubrania

| Przedmiot                           | Cena     |
| ----------------------------------- | -------- |
| Buty                                | 1-10 g   |
| Ciężki płaszcz                      | 1-5 s    |
| Płaszcz ozdobny                     | 1-5 g    |
| Wykwintny płaszcz lub suknia balowa | 10-100 g |
| Zwykły płaszcz lub prosta sukienka  | 1-5 s    |
| Sandały                             | 1-2 s    |
| Koszula, spodnie lub gruba tkanina  | 50-100 b |
| Bawełna                             | 1-2 s    |
| Pościel                             | 5-10 s   |
| Skóra                               | 1-5 g    |
| Jedwab                              | 5-10 g   |
| Futro zimowe                        | 10-20 g  |

### Napoje(jedna porcja)

| Napój                        | Cena     |
| ---------------------------- | -------- |
| Bimber/piwo domowe           | 1-5 b    |
| Cienkie piwo/wino/woda       | 2-10 b   |
| Wino octowe                  | 5-10 b   |
| Wino pospolite               | 10-50 b  |
| Miód pitny                   | 50-100 b |
| Dobre wino/piwo              | 1-3 s    |
| Wykwintne wino/stary rocznik | 5-20 s   |

### Jedzenie(jeden posiłek)

| Jedzenie (jeden posiłek) | Cena (w walutach) |
| ------------------------ | ----------------- |
| Tłuste zlewki            | 1-5 b             |
| Mdłe ale sycące          | 10-20 b           |
| Sycące i smaczne         | 50-100 b          |
| Wyśmienita kuchnia       | 10-20 s           |
| Wystawna uczta           | 10-50 g           |

### Nocleg

| Nocleg (jedna noc)           | Cena (w walutach) |
| ---------------------------- | ----------------- |
| Ławka na wspólnej sali       | 1-5 b             |
| Dzielony pokój czteroosobowy | 10-15 b           |
| Prywatny pokój               | 50-100 b          |
| Prywatny pokój z jedzeniem   | 4-10 s            |
| Apartament                   | 20-50 s           |
| Mieszkanie                   | 1-5 g             |

### Różne

| Różne                       | Cena (w walutach) |
| --------------------------- | ----------------- |
| Dzienna racja żywnościowa   | 1-3 s             |
| Siodło i uzda               | 1-10 g            |
| Stajnia (za noc)            | 1-10 b            |
| Podkucie konia (za podkowę) | 1-5 g             |
| Wełniany koc                | 1-5 s             |

Ceny dostępnego uzbrojenia i pancerzy można znaleźć w dziale Wyposażenie.
